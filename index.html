Crystal Collector

To win the game, match your user score to the random number. If you go over, you lose!

Directions

You will be given a random number at the start of the game. Each crystal below has a given hidden value. Clicking on a crystal will add its value to your total score.

Each time you start a new game a new random number will be generated and the value of each crystal will change.
